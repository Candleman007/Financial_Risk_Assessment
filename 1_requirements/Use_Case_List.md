# Use Case List

## UC-1: User Registration
Actor: User  
Description: User creates a new account using email and password.

## UC-2: User Login
Actor: User  
Description: User logs into the system using valid credentials.

## UC-3: Enter Financial Data
Actor: User  
Description: User enters income, debt, and credit score.

## UC-4: Risk Calculation
Actor: System  
Description: System calculates financial risk using business rules.

## UC-5: ML-Based Risk Prediction
Actor: System  
Description: System predicts risk category using ML model.

## UC-6: View Dashboard
Actor: User  
Description: User views risk score and charts.

## UC-7: Generate Report
Actor: User  
Description: User views risk assessment report.
