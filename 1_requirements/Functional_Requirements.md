# Functional Requirements

FR-1: The system shall allow users to register using email and password.

FR-2: The system shall allow users to log in securely.

FR-3: The system shall allow authenticated users to enter financial details.

FR-4: The system shall calculate financial risk using predefined business rules.

FR-5: The system shall predict risk using a machine learning model.

FR-6: The system shall store user and risk data in a SQL database.

FR-7: The system shall display risk results on a dashboard.

FR-8: The system shall visualize risk distribution using charts.

FR-9: The system shall generate a risk report for users.

FR-10: The system shall log important application events.
