# Non-Functional Requirements

NFR-1: The system shall respond to user requests within 2 seconds.

NFR-2: The system shall ensure secure authentication and password encryption.

NFR-3: The system shall be modular and maintainable.

NFR-4: The system shall support beginner-friendly cloud deployment.

NFR-5: The system shall be scalable for future enhancements.

NFR-6: The system shall log errors and activities for monitoring.

NFR-7: The system shall ensure data consistency and integrity.
